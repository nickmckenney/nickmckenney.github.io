Motley Fool Email Capture

  • Website

This was my first project as an intern at The Motley Fool. Coming onto the team I initially had to learn PHP and understand the team's workflow. Using Lando as a local development environment and docker to host.

After understanding the development environment I used Neverbounce as the 3rd party API to verify emails that users would enter for the newsletter. If it was fake I had to prevent a page refresh and prompt a message to the user. This could only be done using AJAX since the server-side PHP and client-side javascript need to work together.

Home DNS Server

I created a DNS Server for my Home. I initially had to buy a 3rd party modem and router to replace my internet provider's modem router. The previous modem router didn't support replacing the default COX's DNS server which would be an issue in the future. After giving the MAC address of the new modem to COX(my internet provider) I have more control of my new router and the amazing features. With my new server, I installed a Linux server and SSH into it from my desktop.

With Complete control of my server I used a 3rd party DNS server to set up rules that would eventually replace COX's DNS server. If for whatever reason my home server goes offline I can still use the search engine since I have Google's DNS set up as a backup DNS

This small project helped me understand IP routing and learn more about DHCP and DNS than I would from reading my CCNA study guide.
